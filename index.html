Preguntas frecuentes

Toda persona que quiere comenzar con el curso tiene diferentes preguntas, las más comunes son:

¿Qué es CSS?

El código CSS, que junto con el código HTML permite dar forma, color, posición (y otras características visuales) a los documentos web. En resumen, es el "idioma" o idiomas, que un navegador web como Chrome o Firefox conoce y entiende. Nuestro objetivo como diseñadores y programadores web es precisamente ese: aprender el lenguaje y decirle al navegador qué hacer.

¿Qué aprenderé en este curso?

En este curso podrás aprender desde los layout, como el Sass como animar tus proyectos etc... También te ayudara a hacer el paso de los códigos a los frontends, los cuales harán de ti un completo experto en el lenguaje.

¿Qué es CSS GRID?

El CSS GRID és lo que permite dividir una página en regiones o áreas más grandes definiendo el tamaño, la posición y las relaciones de capa entre las partes de control creadas a partir de primitivas HTML.
